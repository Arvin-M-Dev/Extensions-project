### ✨ Features

- ✅ Fully Responsive (Desktop-first)
- ✅ Light & Dark Mode with persistence
- ✅ Advanced Filtering: All / Active / Inactive
- ✅ Dynamic Data Rendering from JSON
- ✅ Toggle Active/Inactive status
- ✅ Remove Extensions
- ✅ State Persistence using localStorage

---

### 🛠️ Tech Stack

- HTML5 Semantic
- CSS3 (Custom Properties, BEM Methodology, Flexbox & Grid)
- Vanilla JavaScript (ES6+)
- Vite as Build Tool
- GitHub Pages for Deployment

---

### 🎯 Key Implementation

**State Management:**
- Dynamic rendering from JSON data
- localStorage for persistence
- Event Delegation for dynamic elements

**Theme Switching:**
- Light/Dark mode toggle
- CSS Custom Properties for theming
- User preference preservation

**Filtering System:**
- All / Active / Inactive tabs
- Real-time filtering
- Dynamic list updates
